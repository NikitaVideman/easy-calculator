# Easy Calculator

## Goals
Purpose of the project is to practice JavaScript, HTML and CSS.

## Features
1. Basic arithmetic operations, `+ - * /`
2. Square roots, `square(n^2)`
3. Raising k to a power of n, `k^n`.
